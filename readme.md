## Next Steps
1.When user select the drink they can see the product's picture and price.

2.When user add the drink to cart they can modify their drink on the detail page.

3.When user checkout they can see the subtotal.

4.When user log in to their account they will see a Welcome message with their name in the navigation bar.

5.When user order on the shop page, they will need to fill out the contact form with the name and phone number/e-mail.If user have logged in to their account the message will be filled out automatically.
